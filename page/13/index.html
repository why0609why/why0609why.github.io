lower_bound和distance函数

这两个函数可以看作比较快速的线性查找。看数组或者vector里面是否存在target。存在返回索引，不存在返回最后一个索引的下一个。

nk两个链表的第一个公共节点

发表于 2020-01-31 | 分类于 牛客

题意

任务



理解

  1. O(m*n)的思路，遍历链表1的每一个节点，去寻找链表2的每一个节点，找到相同的就返回地址值就行。
  2. 仔细想，如果两个链表有公共节点，那么两个链表末尾一定有一部分是相同的(相同地址)，然后其实可以从后同时遍历两个数组，找到两个链表最后一个相同的节点。这里就可以用两个辅助栈，分别模拟两个链表末尾处的节点。找到两个栈的最后相同的栈顶元素就行了。
  3. 仔细发现两个链表的规律，假设两个链表一个长l1，另一个长度为l2，假设l1-l2==m，公共长度是n，然后可以发现这两个链表剩余部分相同，我们这里称为x，那么 l1 = m+x+n ，l2 = x+n，而且从n开始就是两个链表的公共部分，所以只需要让两个指针同时指向两个链表的x开头部分，同时开始向后遍历就行了。而这需要让l1先走m个位置。

nk数组中的逆序对

发表于 2020-01-31 | 分类于 牛客

题意

任务



理解

  1. 用归并排序求逆序对。对于归并排序，采用分治的方法，在merger操作的时候进行逆序对的统计。只要发生后半段的数a[j]比前半段的数小a[i]的时候，那么说明此时前半段那个数到前半段末尾的所有的数都比后半段的那个数大a[j] < a[i…mid]。那么对于j的逆序数就是mid-i+1，如此每次只要发生a[j] < a[i]的时候都进行一波统计逆序数，因为是递归的操作，所有区间的逆序数都会被记录，所以当归并排序结束的时候，逆序对也就全部记录完了。
